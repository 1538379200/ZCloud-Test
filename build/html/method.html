当前可用方法介绍

进入网址

备注

进入网址

driver.get(url);
参数

url – 需要进入的网址

返回类型

None

元素输入

备注

进行元素输入

driver.send_keys(name, val, txt);
参数

  • name – 元素定位方式(xpath、id、name、css等)

  • val – 元素值

  • txt – 需要输入的数据

元素点击

备注

进行元素点击操作

driver.click(name, val);
参数

  • name – 元素定位的方式

  • val – 元素定位值

显性等待

备注

进行显性等待操作，等待直到元素可见可操作

driver.wait_for(name, val, timeout, delay, call_back, *args);
参数

  • name – 必填，元素定位方式

  • val – 必填，元素定位值

  • timeout – 选填，默认10，最大等待时间

  • delay – 选填，默认1，每1秒查看一次

  • call_back – 选填，默认None，可以选择Base实例中的方法，如send_keys、click、report_shot等元素操作

  • args – 选填，当选择的call_back是send_keys时，或者回调的方法有其他参数时，在arg中传入

强制等待

备注

进行强制等待操作，单位为秒，程序将阻塞填入的时间长度

driver.force_wait(timeout);
参数

timeout – 需要等待的时长，整数类型，单位秒

报告截图

备注

对浏览器进行元素截图或者全屏截图并存入报告，如果name或者val未填写，则默认进行全屏截图，否则为元素截图

driver.report_shot(name, val);
参数

  • name – 选填，元素定位方式

  • val – 选填，元素定位值

页面滚动

备注

进行页面滚动，此方法仅封装了滚动到底部和顶部的操作

driver.scroll_top(top);
参数

top – 选填，布尔值，默认False，滚动到底部，为True时滚动到顶部

创建har

备注

创建har，用于捕获网页network接口数据，使用此方法，必须启用了proxy代理

driver.create_har();

获取network响应

备注

获取网页中network的数据，并以列表的形式返回

driver.get_har(filter_url);
参数

filter_url – 字符串类型，指定需要过滤的网址

返回类型

列表

切入iframe

备注

切入到iframe的方法，如果val不填写，默认使用id、name的方式填入iframe，name则变成元素值

driver.switch_iframe(name, val);
参数

  • name – 元素定位方式，当val为空时，name承担val职责

  • val – 选填，元素定位值

切入最后一个句柄

备注

默认切换到最后一个句柄，如果填写了参数，则以参数指定的句柄为准

driver.switch_window_last(hint);
参数

hint – 选填，数字类型，选择当前的第几个句柄

获取元素属性

备注

定位元素并获取其属性值

driver.get_attr(name, val, attr);
参数

  • name – 元素定位方式

  • val – 元素定位值

  • attr – 需要获取的属性名称
